pizza module

class pizza.Pizza(crust, sauces, cheese, toppings)

Bases: object

Represents a single pizza with a crust, sauces, cheese, and toppings.

The cost is calculated at initialization based on selected ingredients.

__init__(crust, sauces, cheese, toppings)

Initialize a new Pizza instance. Calculates cost of pizza upon initialization.

Parameters:

  • crust (str) – The type of crust.

  • sauces (list) – A list of sauces applied to the pizza.

  • cheese (str) – The cheese selection for the pizza.

  • toppings (list) – A list of toppings added to the pizza.
