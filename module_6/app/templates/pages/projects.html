Module 1: Personal Website

The goal of this assignment was to utilize the Flask framework to build a personal portfolio website that can showcase my software projects. The site includes three pages: my biography page, my projects page, and my contact page. Each page was built using the Jinja template engine to streamline development and ensure a consistent visual design across the entire site. The project provided practical experience with Flask, HTML, and CSS, while reinforcing best practices in Git-based version control. Completing this assignment, I now have a scalable website foundation that can be used to highlight future projects or professional milestones.

Skills Used: Python 3.10+, Flask, HTML, CSS, Jinja, GitHub
